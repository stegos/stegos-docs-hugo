Data Directory Layout

Data directory has the following layout:

 1    ├── accounts/  <!-- Accounts
 2    │   ├── 1
 3    │   │   ├── account.pkey <!-- Account #1 public key (address)
 4    │   │   ├── account.skey <!-- Account #2 secret key (address)
 5    │   │   └── history/     <!-- Payment History (RocksDB)
 6    │   └── 2
 7    │       ├── account.pkey <!-- Account #1 public key (address)
 8    │       ├── account.skey <!-- Account #2 secret key (address)
 9    │       └── history/     <!-- Payment History (RocksDB)
10    ├── api.token            <!-- Generated API Token
11    ├── chain/               <!-- Blockchain (RocksDB)
12    ├── network.pkey         <!-- Ephemeral network (consensus) public key.
13    ├── network.skey         <!-- Ephemeral network (consensus) secret key.
