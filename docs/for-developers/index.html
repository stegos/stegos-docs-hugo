Building from Source

Building on Linux and Mac Requirements Stegos requires Rust 1.38.0-nightly (d3f8a0b5d 2019-08-04) to build. The recommended way to install Rust it to use rustup. If you don’t already have rustup, you can install it like this: curl https://sh.rustup.rs -sSf | sh Make sure that these binaries are in your PATH. After that, you should be able to build Stegos from the source. Use nightly-2019-08-04 toolchain when asked. libflint and libgmp - we expect libflint. »
